    4K YouTube Settings

    Internet Speed

    Ensure there is enough upload bandwidth to use 4K streaming.
    Minimum Upload Speed for 4K Streaming is >=30 Mbps

    { width="3.97in", height="1.64in" }
    https://www.speedtest.net/

    YouTube Studio

    • Logged in as Fortitude Barbell
    • Preferred: A channel manager with your own Gmail account
    • YouTube Channel Settings to add a Manager Role

    { width="2.29in", height="1.46in" }
    { width="1.08in", height="1.45in" }

    YouTube Streaming Key

    YouTube requires a 4K enabled Streaming Key

    • https://studio.youtube.com
    • Click Create → Go Live

    { width="4.06in", height="0.95in" }

    Existing 4K Key

    There should be a key already available that allows 4K (2160p) resolution and up to 60 Frames per second

    • Copy the Stream Key for OBS

    { width="4.08in", height="1.77in" }

    Creating a New Key (If Needed)

    ONLY If the key is NOT available perform this step

    • Create a new key with manual settings and enable 2160p

    { width="1.77in", height="1.95in" }

    4K OBS Settings

    OBS Profile Settings

    All the settings below will be stored as a Profile.

    Always create a new profile, so that existing stream profile settings are not lost

    Profiles do not store the Scenes or Sources; these are stored under Scene Collection

    OBS Profiles can be Exported and Imported

    { width="4.07in", height="2.48in" }

    Stream Settings

    • OBS Settings MUST use a stream key for 4K
    • Connect Account option will NOT use a custom 4K Stream Key

    { width="4.08in", height="2.69in" }

    Output Settings

    Output settings will greatly depend on hardware CPU + GPU, these are the minimum settings

    • Output Mode Advanced
    • Audio Track - 1
    • H.264 or H.265 (Hardware)
    • Bitrate -- 20000 Kbps
    • Preset -- Balanced

    4K is not possible on old hardware.

    { width="4.08in", height="2.76in" }

    Audio Settings

    Default must be changed to a lower value for Track 1 to match YouTube recommendations

    • Audio Bitrate - 128

    { width="4.07in", height="0.98in" }

    OBS Canvas

    Set both
    - Base Canvas (Preview) and
    - Output (Streaming)

    to 3840x2160 (4K/2160p) resolution.

    It can be manually typed in if the dropdown box does not show it (Which will be due to physical monitor resolution limitations)

    FPS will be determined by Hardware, 60 is only possible if the hardware is capable
